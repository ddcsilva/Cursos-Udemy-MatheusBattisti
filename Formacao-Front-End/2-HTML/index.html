Aula 3 - Tags sem conteúdo

As tags sem conteúdo são chamadas de tags vazias

Elas não possuem conteúdo e são utilizadas para inserir elementos na página

As tags sem conteúdo são: br, hr e img

Aula 4 - Comentários

Comentários são utilizados para inserir anotações no código

Comentários não são exibidos na página

Comentários são representados pela tag <!-- -->

Aula 5 - Atributos

Google

Os atributos são utilizados para passar informações adicionais para as tags

Os atributos são representados pelo nome do atributo e o valor do atributo

A tag a possui o atributo href que recebe o valor do link. Ex: href="http://www.google.com.br"

Aula 6 - Links em Nova Página

Google

Os links em nova página são representados pelo atributo target="_blank"

Este atributo faz com que o link seja aberto em uma nova aba

Aula 7 - Imagens

As imagens são representadas pela tag img

As imagens possuem o atributo src que recebe o valor do caminho da imagem

Ex: src="img/city.jpg"

Aula 8 - Imagens com Alt

As imagens com alt são representadas pelo atributo alt

Este atributo é utilizado para descrever a imagem

Ex: alt="Noite na floresta"

Aula 9 - Listas não ordenadas

As listas não ordenadas são representadas pela tag ul

As listas não ordenadas possuem itens representados pela tag li

Ex: <li>Carros</li>
